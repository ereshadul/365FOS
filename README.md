# 365FOS — CRM & Data Analytics Learning Project

End-to-end hands-on training project simulating a real Data Analyst / CRM Admin role  
at a magnet sales company (RonTech Magnets).

## 🛠️ Tools Covered
| Tool | Purpose |
|---|---|
| Microsoft 365 | Core platform & Outlook |
| Dynamics 365 CRM | Customer & pipeline management |
| Power Automate | Workflow automation |
| Power BI | Dashboards & reporting |
| Excel | Data cleaning & analysis |
| Microsoft Copilot | AI-assisted reporting |
| Claude AI | Narrative analysis & insights |

## 📁 Project Phases
- **Phase 0** — Setup & Pre-Work
- **Phase 1** — Excel Data Cleaning & Analysis
- **Phase 2** — D365 CRM Build
- **Phase 3** — Outlook Integration
- **Phase 4** — Power Automate Flows
- **Phase 5** — Power BI Dashboard
- **Phase 6** — AI-Powered Reporting
- **Phase 7** — Final Integration & Interview Scenarios

## 🎯 Goal
Complete all phases to simulate a real analyst role and be interview-ready  
to demo every tool live with confidence.

## ⏱️ Estimated Completion
27–43 hours total across 2–3 weeks
